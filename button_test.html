Buttons that work and can open in a new tab

Summary

  • In order to have the option to open a link in a new tab (using right click), one must use the a href tag

  • The 'button' tag does not do anything by default. Only styles it like a button

  • Therefore, the button tag needs a form tag and action-"link" attribute OR formaction="link" attribute OR a href to be clickable

  • form and formaction="link" attributes will add a ? at the end of the url, unless using method='get'

Disclaimer: each link can be defaulted to open in a new tab with target='_blank' as an attribute
